# Character-Level Language Model (Bigram + Neural Network)

This project implements a **character-level language model** for generating names using two approaches:

1. **Statistical Bigram Model**
2. **Neural Network (trained with Negative Log Likelihood)**

The goal is to demonstrate how a neural network can learn the same probability distribution as a simple statistical model.

---

## 📌 Overview

Given a dataset of names, the model learns:

> **P(next character | current character)**

It then generates new names by sampling from this learned distribution.

---

## ⚙️ Implementations

### 1. Bigram Model (Statistical)

* Splits words into character pairs (bigrams)
* Counts occurrences of each pair
* Converts counts into probabilities

```text
text → bigrams → counts → probabilities → sampling
```

✔️ No training required
✔️ Direct computation from data

---

### 2. Neural Network Model

* Uses one-hot encoding of characters
* Learns a weight matrix `W`
* Outputs logits → converted to probabilities using softmax
* Trained using **Negative Log Likelihood (NLL)**

```text
input → one-hot → linear layer → logits → softmax → probabilities
```

✔️ Trained with gradient descent
✔️ Learns probabilities instead of counting them

---

## 🔍 Key Insight

After training, the neural network learns a distribution that closely matches the bigram probability matrix:

```python
probs_nn = torch.softmax(W, dim=1)
print((probs_nn - P).abs().mean())
```

This shows that:

> A neural network can learn the same probability distribution as a statistical model.
